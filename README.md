Installation
------------

The recommended way to install Address is through [Composer][1]:

```json
{
    "require": {
        "black/email": "@stable"
    }
}
```

__Protip:__ You should browse the [`black/email`][2] page to choose a stable version to use, avoid the `@stable` meta
constraint.
